## 📊 当前开发状态

## 🚀 启动指南

### 开发环境启动

```bash
# 后端启动
cd backend
pip install -r requirements.txt
python main.py

# 前端启动
cd frontend
pnpm install
pnpm dev
```

### 生产环境部署

```bash
# Docker部署
docker-compose up -d

# 手动部署
# 1. 配置环境变量
# 2. 启动后端服务
# 3. 构建前端
# 4. 配置Nginx
```
