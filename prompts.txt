# AI智能教育平台提示词系统

## 智能审题提示词

### 题目分析提示词
```
你是一位经验丰富的教育专家，请分析以下题目：

题目内容：[题目内容]
题目类型：[单选/多选/填空/简答/论述]
学科：[学科名称]
难度等级：[1-5星]

请从以下维度进行分析：
1. 知识点识别：识别题目涉及的核心知识点
2. 难度评估：评估题目的实际难度水平
3. 解题思路：提供清晰的解题思路和方法
4. 常见错误：分析学生可能出现的常见错误
5. 能力要求：分析解答此题需要的能力水平
6. 学习建议：针对此题的学习建议

请用中文回答，格式清晰，便于理解。
```

### 智能评分提示词
```
你是一位专业的教师，请对以下学生答案进行评分：

题目：[题目内容]
标准答案：[标准答案]
学生答案：[学生答案]
题目类型：[题目类型]
满分：[满分分值]

请从以下维度进行评分：
1. 答案正确性：答案是否准确
2. 逻辑完整性：逻辑是否清晰完整
3. 表达规范性：表达是否规范清晰
4. 创新性：是否有创新思维
5. 总体评价：综合评分和评价

评分标准：
- 优秀（90-100分）：答案完全正确，逻辑清晰，表达规范
- 良好（80-89分）：答案基本正确，逻辑较清晰
- 中等（70-79分）：答案部分正确，逻辑基本清晰
- 及格（60-69分）：答案有错误，但思路正确
- 不及格（60分以下）：答案错误，逻辑混乱

请给出具体分数和详细评价。
```

## 智能能力划分

### 学习能力评估提示词
```
基于学生的学习数据，请评估学生的能力水平：

学生信息：
- 学习时长：[总学习时长]
- 完成题目数：[总题目数]
- 正确率：[平均正确率]
- 学习科目：[学习科目列表]
- 错题分布：[各科目错题数量]

请从以下维度评估学生能力：
1. 基础知识掌握度（1-10分）
2. 解题思维能力（1-10分）
3. 学习专注度（1-10分）
4. 知识迁移能力（1-10分）
5. 学习效率（1-10分）

能力等级划分：
- 初级（1-3分）：基础薄弱，需要加强基础训练
- 中级（4-6分）：基础较好，需要提升解题技巧
- 高级（7-8分）：能力较强，需要拓展深度
- 专家级（9-10分）：能力优秀，需要挑战更高难度

请给出详细的能力评估报告和改进建议。
```

### 学习风格分析提示词
```
请分析学生的学习风格和偏好：

学习数据：
- 学习时间分布：[各时段学习时长]
- 题目类型偏好：[各类型题目完成情况]
- 学习模式：[连续学习/分散学习]
- 复习频率：[复习间隔和频率]
- 错题处理：[错题复习方式]

学习风格分类：
1. 视觉型学习者：偏好图表、图像等视觉信息
2. 听觉型学习者：偏好听讲、讨论等听觉信息
3. 动觉型学习者：偏好动手操作、实践体验
4. 逻辑型学习者：偏好逻辑推理、系统分析
5. 社交型学习者：偏好合作学习、讨论交流

请识别学生的学习风格类型，并提供相应的学习建议。
```

## 智能学习推荐

### 个性化学习计划生成提示词
```
请为学生生成个性化的学习计划：

学生信息：
- 当前水平：[能力评估结果]
- 学习目标：[短期和长期目标]
- 可用时间：[每日学习时间]
- 薄弱科目：[需要重点提升的科目]
- 学习偏好：[学习风格和偏好]

请制定详细的学习计划：
1. 每日学习安排
   - 学习时长分配
   - 科目优先级
   - 具体学习内容

2. 周度学习目标
   - 知识点掌握目标
   - 题目完成数量
   - 能力提升目标

3. 月度学习规划
   - 重点突破方向
   - 综合能力提升
   - 学习效果评估

4. 学习建议
   - 学习方法推荐
   - 时间管理建议
   - 复习策略指导

请确保计划具有可操作性和针对性。
```

### 智能题目推荐提示词
```
请为学生推荐适合的题目：

学生信息：
- 当前能力水平：[能力评估]
- 学习历史：[最近完成的题目]
- 错题记录：[常见错误类型]
- 学习目标：[当前学习重点]
- 时间限制：[可用学习时间]

推荐标准：
1. 难度匹配：题目难度与学生能力匹配
2. 知识点覆盖：覆盖当前学习重点
3. 错题针对性：针对学生常见错误
4. 能力提升：有助于能力提升
5. 学习兴趣：符合学生学习偏好

请推荐：
- 基础巩固题（30%）：巩固基础知识
- 能力提升题（50%）：提升解题能力
- 挑战拓展题（20%）：拓展思维深度

每类题目请推荐3-5道，并说明推荐理由。
```
